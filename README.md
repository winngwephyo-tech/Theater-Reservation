# Theater-Reservation
Technologies
1.PHP 7.4.21
2.Laravel 7.28.4
3.Composer 2.1.3
4.MySQL 8.0.25

Development
1.Create .env(copy contents for .env.example)
2.Run the following
$composer install
composer dumpautoload -o

$php artisan key:generate
$php artisan migrate:install
$php artisan migrate
$php artisan db:seed
$php artisan serve
